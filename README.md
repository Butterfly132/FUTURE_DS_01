# Business Sales Performance Dashboard

## Project Overview
This project analyzes business sales data to identify revenue trends, top-selling products, geographic performance, and customer insights using Power BI.

## Tools Used
- Power BI
- Online Retail Dataset

## Key Analysis
- Revenue trend over time
- Top revenue-generating products
- Revenue by country
- Top customers by revenue

## Key Insights
- A small number of products generate most revenue.
- United Kingdom contributes majority of sales.
- Certain customers generate significantly higher revenue.
